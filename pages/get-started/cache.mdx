# Cache queries in ReadySet

Once your application is sending SQL queries through ReadySet, you can see which queries are cacheable. 

You can run diagnostics with both Postgres and MySQL to ascertain costly queries. This information might 
be helpful in determining which queries you should cache with ReadySet.
